# Документ фиксации текущего состояния платформы (AS-IS)

## 1. Software Architecture Canvas: Бизнес-контекст и цели

| Категория | Описание |
| :--- | :--- |
| **Бизнес-цели** | <ul><li>Обеспечение бесперебойного оформления заказов в режиме 24/7.</li><li>Поддержка многоканальности (Сайт, Маркетплейсы, Мобильное приложение).</li><li>Гибкость ценообразования и промо-акций.</li><li>Быстрая адаптация под новых логистических партнеров.</li></ul> |
| **Стейкхолдеры** | <ul><li>**Внутренние**: Бизнес-заказчики (маркетинг, операции), Разработчики (Java, .NET, Python), SRE/DevOps, Поддержка (операторы колл-центра).</li><li>**Внешние**: Покупатели, Продавцы (селлеры), Логистические партнеры, Платежные шлюзы.</li></ul> |
| **Ключевые требования** | <ul><li>**Функциональные**: Создание заказа, Расчет итоговой цены (с учетом промо и региона), Резервирование доставки, Оплата, Управление статусами.</li><li>**Нефункциональные**: Высокая доступность (особенно в часы пик), Жесткая согласованность данных (ACID), Наблюдаемость процессов.</li></ul> |
| **Ограничения** | <ul><li>Технический долг в Oracle PL/SQL.</li><li>Высокая связанность компонентов (сильная зацепленность).</li><li>Смесь синхронных и асинхронных протоколов.</li><li>Отсутствие единой модели данных.</li><li>Частые ручные вмешательства в бизнес-процессы.</li></ul> |
| **Ключевые риски** | <ul><li>Отказ платежного шлюза или логистики в момент оформления заказа.</li><li>Несовпадение данных в отчетах (отсутствует Single Source of Truth).</li><li>Сложность внесения изменений и высокий риск регрессионных ошибок.</li><li>Блокировки базы данных в периоды крупных распродаж.</li></ul> |
| **Проблемные зоны** | <ol><li>**Расчет цены**: Логика размазана по коду, внешнему Promo-сервису и Oracle-функциям.</li><li>**Статусы**: Хаотичный жизненный цикл заказа с ручными правками.</li><li>**Интеграции**: Синхронные вызовы в платежи и логистику блокируют основной поток оформления.</li><li>**База данных**: «Длинные» транзакции и интеграция систем между собой через общие таблицы в Oracle (с .NET).</li></ol> |
